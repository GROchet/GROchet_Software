Bienvenue sur le GitHub du GROchet!

Le GROchet est une machine à pince open source de laquelle il est possible de changer les niveaux de difficulté. Il a été conçu dans le cadre du projet de quatrième session en génie robotique à l'Université de Sherbrooke.
